# Stage 3 原型文档

> 艺术成长管理APP - 第三期功能原型
>
> **目标**: 完成报名-付费-交付的完整商业链路

## 📋 概述

Stage 3 是艺术成长管理APP的第三期功能，主要完成商业闭环，包括在线支付、课程报名、完整学员档案、老师端完整功能以及营销工具。

## 🎨 设计特点

- 🔄 **流程清晰**: 支付流程、报名流程都有明确的步骤指示

## 💡 核心交互流程

### 1. 课程报名支付流程
```
选择课程 → 选择学员 → 选择优惠券 → 确认订单 →
选择支付方式 → 扫码支付 → 支付成功 → 开始学习
```

**关键功能点:**
- 三步进度指示器
- 优惠券弹窗选择
- 价格自动计算
- 微信/支付宝支付切换
- 支付倒计时
- 支付成功页面

### 2. 学员档案查看流程
```
进入档案 → 查看学习概览 → 切换标签页
(课程记录/成长轨迹/荣誉成就) → 导出档案
```

**关键功能点:**
- 多标签切换
- 能力雷达图可视化
- 成长时间线
- 学习打卡日历
- 多格式导出选项

### 3. 老师评审作品流程
```
查看待评作品列表 → 点击评审 → 查看作品详情 →
评分 → 填写三段式反馈 → 提交评审 → 通知学员
```

**关键功能点:**
- 待评作品列表
- 实时通知提醒
- 班级数据对比
- 学员成绩分布

### 4. 营销活动创建流程
```
选择工具类型(优惠券/拼团/推荐) → 填写活动信息 →
设置规则 → 发布活动 → 查看数据 → 分享推广
```

**关键功能点:**
- 三种营销工具切换
- 优惠券样式预览
- 拼团倒计时
- 邀请码生成
- 社交分享

## 📊 数据展示亮点

### 学员档案
- **能力雷达图**: 六维度能力评估（构图、色彩、技法、创意、完成度、观察力）
- **学习打卡**: 周历展示，直观显示学习连续性
- **成长时间线**: 关键里程碑和成就展示

### 老师工作台
- **作品提交趋势**: 柱状图展示每日提交量
- **成绩分布**: 百分比展示各分数段占比
- **班级对比**: 横向对比不同班级平均分
- **本月之星**: 排行榜展示优秀学员

### 营销工具
- **优惠券数据**: 发行数、领取数、使用数、使用率
- **拼团数据**: 成团数、参与人数、成交额、成团率
- **推荐数据**: 累计邀请、成功注册、获得奖励

## 📱 移动端优化

### 触摸优化
- 按钮最小点击区域: 44x44px
- 卡片间距适合手指操作
- 表单输入框大小适中

### 布局优化
- 移动端自动切换为单列布局
- 侧边栏在移动端收起
- 底部操作栏固定在底部
- 模态框从底部滑入

### 性能优化
- 使用CSS动画而非JS动画
- 图片使用渐变色占位
- 按需加载内容

## 🎨 设计规范

### 颜色
- **主色**: Blue-600 (#3B82F6)
- **辅助色**: Purple-600, Green-600, Orange-600
- **成功**: Green-500
- **警告**: Orange-500
- **错误**: Red-500
- **中性**: Gray-500

### 间距
- **基础**: 4px (Tailwind的1单位)
- **小**: 8px (2)
- **中**: 16px (4)
- **大**: 24px (6)
- **超大**: 32px (8)

### 圆角
- **小**: 8px (rounded-lg)
- **中**: 12px (rounded-xl)
- **大**: 16px (rounded-2xl)
- **圆形**: 50% (rounded-full)

### 阴影
- **小**: shadow-sm
- **中**: shadow
- **大**: shadow-lg
