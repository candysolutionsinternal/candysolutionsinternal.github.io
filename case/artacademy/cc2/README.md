# 美术成长管理APP - 第二期原型

## 概述

这是美术成长管理APP第二期（Stage 2）功能的交互式HTML原型，展示了在MVP基础上新增的成长可视化功能。

**原型版本**: Stage 2 (第二期)
**创建日期**: 2026-01-25
**技术栈**: 纯 HTML + CSS + JavaScript (无框架)

## 文件结构

```
prototype/stage1/
├── index.html                        # 导航页面（入口）
├── prototype-stage2-student.html     # 学生端原型
├── prototype-stage2-teacher.html     # 老师端原型
└── README.md                         # 本文档
```

## 快速开始

### 方式一：直接打开（推荐）

1. 在浏览器中打开 `index.html`
2. 选择查看学生端或老师端原型

### 方式二：本地服务器

```bash
# 在 prototype/stage1 目录下启动本地服务器
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000

# 访问: http://localhost:8000
```

## 技术实现

### 前端技术
- **HTML5**: 语义化标签
- **CSS**: Tailwind CSS (via CDN)
- **JavaScript**: 原生 ES6+，无框架依赖

### 外部依赖（CDN）
- **Tailwind CSS**: 快速样式开发
- **Chart.js v4.4.1**: 雷达图、折线图
- **html2canvas v1.4.1**: 分享图片生成

### 核心功能实现

#### Tab 切换
```javascript
// 监听 tab 按钮点击，动态切换内容区域
tabBtns.forEach(btn => {
    btn.addEventListener('click', () => {
        // 更新按钮状态和显示对应内容
    });
});
```

#### 模态窗口
- 通知中心模态窗口
- 分享报告模态窗口
- 学生详情模态窗口
- 批改反馈模态窗口

#### 数据可视化
```javascript
// 雷达图：展示5维能力评估
new Chart(radarCtx, {
    type: 'radar',
    data: { ... },
    options: { ... }
});

// 趋势曲线：8周进步趋势
new Chart(trendCtx, {
    type: 'line',
    data: { ... },
    options: { ... }
});
```

#### 分享图片生成
```javascript
html2canvas(element).then(canvas => {
    const link = document.createElement('a');
    link.download = '成长报告.png';
    link.href = canvas.toDataURL();
    link.click();
});
```
